# UI Base

Componentes base del design system inicial de DAFREQ.

## Button

```tsx
<Button variant="primary">Guardar</Button>
<Button variant="danger">Eliminar</Button>
<Button variant="warning">Desactivar</Button>
<Button variant="success">Reactivar</Button>
<Button variant="outline">Editar</Button>
<Button variant="ghost">Cancelar</Button>
```

### Variantes

- `primary`
- `secondary`
- `danger`
- `warning`
- `success`
- `ghost`
- `outline`

### Tamaños

- `sm`
- `md`
- `lg`
- `icon`

## StatusBadge

```tsx
<StatusBadge variant="active">Activo</StatusBadge>
<StatusBadge variant="inactive">Inactivo</StatusBadge>
<StatusBadge variant="primary">En curso</StatusBadge>
<StatusBadge variant="warning">Pendiente</StatusBadge>
<StatusBadge variant="success">Completado</StatusBadge>
```
